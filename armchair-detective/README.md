安楽椅子探偵
====================


## ゲーム概要

山荘で殺人事件が発生しました。そのとき山荘にいたのは、プレイヤーの助手、被害者、ほか3名の合計5名です。プレイヤーは助手にチャットで助言や指示を送ることで、助手と協力して事件の解決に挑みます。
